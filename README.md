This is a real-time chat application built using Spring Boot and WebSocket technology.
It allows multiple users to communicate instantly through live messaging without page refresh.

The application demonstrates real-time communication using WebSocket protocol and STOMP messaging.

Features:
-> Real-time messaging
-> Multiple users support
-> WebSocket integration
-> STOMP protocol messaging
-> Live message broadcasting
-> Clean UI interface
-> Join/Leave notifications

Tech Stack:
-> Java
-> Spring Boot
-> Spring WebSocket
-> STOMP Protocol
-> HTML
-> CSS
-> JavaScript
-> Maven
